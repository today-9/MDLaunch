**MDLaunch** は、フォルダに置いたただの `.md` ファイルを Notion ライクな UI で眺める・探す・増やすためのローカルツールです。

## 使い方

- ノートの実体は `vault/` フォルダ内の Markdown ファイルです
- **`Ctrl + K`** でどこからでも検索できます
- 「＋ 新規ノート」で作成すると、そのまま **VSCode が開きます**
- 編集して保存すれば、数秒でこの画面にも反映されます

詳しくは [[記法ガイド]] を見てください。サンプルとして [[サンプル ロードマップ]] も置いてあります。

## なぜファイルベース?

> データがただのテキストファイルなら、どんな環境でも壊れない。

- VSCode でも Git でも grep でも扱える
- 会社の環境でもクラウド不要で動く
- ツールを捨てても資産(ノート)は残る
